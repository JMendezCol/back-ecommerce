# 🛍️ Back Ecommerce Java

Backend del proyecto desarrollado en Java con Spring Boot.  
Proporciona endpoints REST para la gestión de productos, usuarios, autenticación y pedidos.

---

## 🚀 Requisitos

- **Java 17** o superior
- **Maven 3.9+**
- **MySQL**
